# 📋 Console ToDo Manager (JSON-based)

![Python](https://img.shields.io/badge/python-3.x-blue.svg)
![Storage](https://img.shields.io/badge/storage-JSON-green.svg)

Простой, но эффективный консольный менеджер задач на Python, который сохраняет ваши дела в локальный файл. Этот проект демонстрирует базовую работу со структурами данных, обработку исключений и персистентность (сохранение) данных между запусками программы.

---

## ✨ Функционал

* **Создание задач:** Каждая задача содержит текст, дедлайн, приоритет и автоматический статус активности (`True`).
* **Показ всех задач:** Считывание актуального списка задач прямо из файла.
* **Интерактивное обновление статуса:** Возможность отметить задачу как выполненную (статус меняется на `False`) по её названию.
* **Надежное хранение:** Использование формата JSON. Данные не стираются при закрытии консоли.
* **Устойчивость к ошибкам:** Защита от отсутствия файла или его повреждения (`FileNotFoundError`, `JSONDecodeError`).

---

## 🛠️ Как это устроено "Под капотом"

В коде используются правильные практики для начинающего Python-разработчика:

1. **Контекстные менеджеры (`with open(...)`):** Гарантируют, что файл закроется автоматически, даже если в процессе чтения или записи произойдет сбой.
2. **Безопасное чтение JSON:** ```python
   try:
       with open("task_list.json", "r", encoding="utf-8") as file:
           list_tasks = json.load(file)
   except (FileNotFoundError, json.JSONDecodeError):
       list_tasks = []
